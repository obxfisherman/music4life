Music For Life notes
1.10.17

Sevice starts at Rpi boot

To stop the service:
sudo /etc/init.d/myservice.sh stop

To start the service:
sudo /etc/init.d/myservice.sh start

To check status
sudo /etc/init.d/myservice.sh status

To resart:
sudo /etc/init.d/myservice.sh restart

To see console output look at the log file at
/tmp/m4l.log

To see program metrics look in:
/home/pi/share/music4life/logs
